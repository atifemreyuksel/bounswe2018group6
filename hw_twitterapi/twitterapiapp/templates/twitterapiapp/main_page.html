Find Following List of a User

Web interface for API call: {% include "twitterapiapp/server" %}/followings/screen_name?count=20

API call: {% include "twitterapiapp/server" %}/api/followings/screen_name?count=20

Gets the list of users that the user with given screen name follows. By default, gets last 20 users, however custom number can be set using (optional) 'count' parameter.
